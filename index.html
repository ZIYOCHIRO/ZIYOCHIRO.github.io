Creating a Mac version of Your iPhone App

Posted on 2019-12-05 | Post modified: 2019-12-05

Link:
Requirements:

  • Xcode11 and later
  • MacOS 10.15 and later

I have a simple iOS App that only have a viewController and a few buttons and lables on it. It looks like this:
Imgur
If you want to create a Mac version of it without changing any code, you can do this by using Mac-Catalyst. It’s fast and simple.

#1. Pre-Setting
###Step1
Check the Mac option in Target Device.
Imgur
###Step2
Normally, the code can just run on you Mac fine, but you might encounter this situation that you cannot run the code on you Mac because you have to add some Mac’s architecture in the Build Setting.
Imgur
You can fix this by manually add ‘i386’ ‘x86_64’ to the Valid Architectures.
Imgur
